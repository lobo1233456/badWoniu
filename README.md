#### 这是什么？

这是收集自互联网的，[蜗牛学院](http://www.woniuxy.com/)的一些槽点和黑点。

#### 目的

展示和呈现一个更真实而全面的蜗牛学院，值得一提的是，有槽点和负面的声音是非常正常事情。这并不代表蜗牛学院不好，仅能证明在选择蜗牛学院的时候，应该更多一点谨慎。
